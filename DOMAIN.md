## 免费域名能到什么程度？

| 方案 | 地址示例 | 费用 | 说明 |
|------|----------|------|------|
| GitHub Pages（当前） | `leonardlxl.github.io/eona/` | 免费 | 已是最简单的免费方案之一 |
| Vercel | `eonalab.vercel.app` | 免费 | 推荐；`eona.vercel.app` 已被别人占用 |
| Cloudflare Pages | `eona.pages.dev` | 免费 | 需在 cloudflare.com 部署同一仓库 |
| 买自己的域名 | `eona.com` / `eona.style` | 约 $10–15/年 | **最短、最专业**，可绑 GitHub 或 Vercel |

**完全免费又很短**（如只要 `eona.com`）基本做不到——短域名需要购买。

## 绑定自己的域名（买了域名之后）

### GitHub Pages

1. 仓库 → **Settings** → **Pages** → **Custom domain**
2. 填 `shop.eona.com` 或 `www.eona.com`
3. 在域名注册商添加 DNS（GitHub 会提示 CNAME / A 记录）

### Vercel

1. 项目 → **Settings** → **Domains** → **Add**
2. 按提示配置 DNS

**托管仍免费**，只付域名年费。
