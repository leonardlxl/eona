# Eona — 分享链接 / Share URLs

`*.vercel.app` 在部分网络（尤其国内运营商、公司 WiFi）可能打不开，**不是你的网站坏了**。

## 推荐分享顺序

| 优先级 | 链接 | 适合 |
|--------|------|------|
| 1 | https://leonardlxl.github.io/eona/ | 国内、公司网络、手机流量 |
| 2 | https://eonalab.vercel.app | 海外、稳定 VPN |
| 3 | https://eona-three.vercel.app | Vercel 备用域名 |

三个地址内容相同，任选一个能打开的即可。

## 给别人发链接时

**国内朋友 / 不确定网络：** 直接发 GitHub 镜像

```
https://leonardlxl.github.io/eona/
```

**海外朋友：** 可发 Vercel

```
https://eonalab.vercel.app
```

## 自己排查

1. 手机 **关 WiFi，用 4G/5G** 试一次  
2. **换浏览器** 或无痕模式  
3. 开/关 VPN 各试一次（不同网络对 Vercel / GitHub 表现相反）  
4. 两个链接都试：GitHub 镜像 + Vercel  

## 长期方案（可选）

- 购买自己的域名（如 `eona.style`），绑到 Vercel 或 GitHub Pages，通常比免费子域名更稳定  
- 见 `DOMAIN.md`
